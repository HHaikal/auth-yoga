# simple authentication with graphql yoga

## Start
> npm install -g nodemon

> npm install / yarn
- create yoga_auth database
> sequelize db:migrate

> nodemon index.js

> localhost:4000

Lib:
- Graphql-yoga
- bcryptjs
- jsonwebtoken
- sequelize ( ORM )
- mysql2
- dotenv
- apollo-server ( error handling )
